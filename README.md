## 🚀 Key Features & Visual Impacts

### 🟢 Live Signal State Visualization
* Real-time rendering dynamically colors node paths based on true/false conditions.
* High states activate a vibrant **Neon Green (`#10B981`)** glow, while low states drop back to clear **Slate Gray (`#94A3B8`)**.
* Output lightbulb nodes feature a glowing pulsating animation loop when powered.

### 📐 Dynamic Node Pin Alignment
* Connectors automatically adjust to match exact chip architectures (e.g., 3 inputs/2 outputs for Full Adders).
* Smooth SVG transitions apply during hardware state switches.

### 🔍 Mouse Hover Event: Smart Chip Pinout Popup
* Hovering directly over any complex integrated circuit (IC) triggers an instant **Glassmorphism Detail Popup**.
* Seamless layout detection ignores handle zones to avoid obstructing active drag-and-drop wiring paths, automatically sliding out a pin configuration map and functional description.

### 🧩 Drag-and-Drop & Smart Gate Transformation
* Freely arrange logic gates on an infinite canvas.
* Attaching a NOT gate to basic gates (AND, OR, XOR) triggers an automatic morphing effect, transforming the connection into a compact dot representation (NAND, NOR, XNOR).
