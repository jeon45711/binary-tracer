# Binary Tracer

An intuitive, web-based logic gate simulator designed to make learning digital electronics easy and interactive. It features a network-like structure inspired by Cisco Packet Tracer, allowing users to build, simulate, and analyze logic circuits seamlessly.

## Key Features

* **Drag-and-Drop Interface:** Easily build complex circuits by dragging components onto the canvas.
* **Smart Gate Transformation:** Attaching a NOT gate to an AND, OR, or XOR gate automatically converts it into a compact dot representation (NAND, NOR, XNOR).
* **Circuit Component Inspection:** Hover over pre-made circuits to instantly view their internal logic gate composition (AND, OR, NOT).
* **Live Visual Simulation:** Real-time state visualization clearly shows active and inactive signal paths.
* **Multi-Platform Design:** Built with a web-first approach, with a structural architecture ready for future mobile application integration.

## Additional Backend Features

* **Lobby Web Server:** Integrated web server supporting user lobby management.
* **Cloud File Storage:** Save and load circuit designs on the web, with data architecture ready for future mobile app synchronization.

## Supported Components

* **Basic & Complex Gates:** AND, OR, NOT, NAND, NOR, XOR, XNOR
* **Pre-made Circuits & Blocks:**
  * Half Adder & Full Adder
  * MUX (2-to-1 Multiplexer)
  * D Flip-Flop
  * Clock (CLK) Generator
  * 7-Segment Decoder
